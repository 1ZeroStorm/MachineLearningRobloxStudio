# Looking For the Best silhouette score by using K-Means Clustering Algorithm to generate randomized rocks

## How to run
1) Download all files (except this README file)
2) Open roblox studio, create a new workspace
3) insert/import ```RemoteEvents``` to ReplicatedStorage inside the explorer tab
4) insert/import ```KmeansGraphGUI.rbxm``` to StarterGUI inside the explorer tab
5) insert/import ```KmeansTrig.lua``` to StarterPlayer -> StarterPlayerScripts inside the explorer tab
6) insert/import ```kmeans_ver2.lua``` to ServerScriptService inside the explorer tab
